Interactive Streamlit app to explore Harvard Art Museum artifacts. Users can collect JSON data, insert it into a MySQL database, view metadata, media, and colors, and run predefined SQL queries for detailed analysis.  

## Features

- Collect artifact data from JSON files.
- Preview artifact metadata, media, and colors in JSON format.
- Insert data into MySQL with duplicate handling.
- Scrollable SQL table previews for Metadata, Media, and Colors.
- Run predefined queries for insights such as:
  - Top cultures
  - Artifacts by century
  - Artifacts with specific hues or missing data
- User-friendly Streamlit interface with centered titles and responsive layout.
